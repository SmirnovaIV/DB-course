Данный комплект содержит инсталляционный файл системы управления сайтами UMI.CMS Commerce Encoded, предназначенной для ознакомления на локальном компьютере под Microsoft Windows. Для установки требуется подключение к интернету (рекомендуется высокоскоростное). Система будет работать на основе пакета «Денвер», заранее настроенного и проверенного производителем.

Внимание! При запуске сервера создается (временно) виртуальный диск "U:". Для корректной работы необходимо, чтобы эта буква диска была свободна. После остановки сервера буква диска становится вновь свободной.

Чтобы начать работать с UMI.CMS, запустите сервер (ярлык запуска в стартовом меню или на рабочем столе). При работающем сервере вы в любой момент сможете открыть демо-сайт по ссылке http://localhost

При первом запуске система предложит вам попробовать в работе инсталлятор UMI.CMS, чтобы вы смогли увидеть, как просто устанавливается система на хостинге. При последующих запусках будет открываться демо-сайт. В процессе установки система запросит параметры соединения с БД. Введите следующие параметры:

	Хост:    localhost
	Логин:   umi
	Пароль:  umi
	Имя БД:  umi

После установки вы сможете зайти в раздел администрирования сайта по адресу:
http://localhost/admin/
с логином и паролем администратора, которые вы указали в процессе установки.

Не забывайте корректно завершать работу сервера (Ярлык для остановки сервера находится в меню «Пуск» или на рабочем столе)! Это важно для сохранности данных в базе MySQL.


Если вы захотите повторно запустить установку UMI.CMS на сайт, необходимо удалить из директории сайта файлы «installed» и «.htaccess».
